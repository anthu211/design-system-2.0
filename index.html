Spacing & Grid

Spacing scale, layout grid, and component padding guidelines for the desktop cybersecurity dashboard. All values are multiples of the base unit (4px).

Base unit: 4px. All spacing values are multiples of 4. This ensures consistent vertical rhythm and alignment across the dashboard layout.

Target Resolutions

Prevalent AI is a desktop-only application. Mobile and tablet viewports are not supported. The UI is optimised for large displays used in security operations centres (SOC) and analyst workstations.

Desktop-only product. No responsive breakpoints for mobile or tablet. The application renders at a minimum of 1280px and is optimised for 1440px and 1920px wide displays common in SOC environments.

Design Canvas Standard
All Figma frames should be designed at 1440 × 1024px as the canonical canvas. This represents the most common analyst workstation display at standard browser zoom. Components are designed at 1x and use @2x asset exports for retina clarity.
